BONUS GALLERY IMAGE FOLDERS

Keep these five language folders:
  public/bonuses/gallery/fr/
  public/bonuses/gallery/de/
  public/bonuses/gallery/it/
  public/bonuses/gallery/nl/
  public/bonuses/gallery/pl/

Delete (if present):
  public/bonuses/gallery/shared/
  any image files directly inside public/bonuses/gallery/
  any other folders inside public/bonuses/gallery/

The BonusStackGallery component loads images from:
  /bonuses/gallery/{fr|de|it|nl|pl}/{filename}
